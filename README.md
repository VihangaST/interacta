Interacta - An AI-powered quiz app for businesses to promote their products in an engaging way

❓ Are you still creating quiz questions manually in Excel and uploading them?
It’s inefficient, time-consuming, and not user-friendly.

✨ With Interacta, shop owners can provide a product name, description, desired number of questions, and an image. AI instantly generates a complete quiz set with questions, options, and answers.

💡 Not just products , Interacta also allows creating quiz sets on any domain or topic by entering the subject and number of questions. This flexibility makes it useful for various scenarios, from product promotions to knowledge-based quizzes.

👥 Customers can join quizzes, explore new arrivals, answer questions, and even win rewards by turning product promotion into a fun, interactive experience.

🔹 Smarter product promotion
🔹 Automatic AI-generated quizzes
🔹 Flexible quiz creation on any domain
🔹 Interactive engagement with customers

🔧 Tech Stack: MERN (MongoDB, Express.js, React, Node.js) + Boostrap + LLM API Integration
